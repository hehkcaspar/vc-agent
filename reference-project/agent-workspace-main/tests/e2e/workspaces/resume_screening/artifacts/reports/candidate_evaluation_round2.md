1. **AI tool mentions in resume = immediate attention** (LangChain, LLM APIs, n8n, Dify, Coze)
2. **Quantifiable metrics matter** (efficiency gains, automation rates, user numbers)
3. **Live product URLs > theoretical projects**
4. **Guangzhou location is non-negotiable**
5. **Younger candidates (0-2yr) fit culture better than seniors**

### Red Flags to Watch:
1. **Location mismatch** - automatic rejection
2. **Salary expectations above budget** - waste of time
3. **Design-only portfolios** for growth/design roles
4. **Enterprise backgrounds** without startup experience
5. **Overqualification** - may resist fast-scratch culture

### Process Improvements:
1. **Request portfolio links upfront** for design/growth roles
2. **Verify location early** in screening call
3. **Ask for live demos** during technical interview
